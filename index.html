About

The purpose of this application is for users to keep track of planned activities, along with searching for new ones.

  • We utilize the Bored API, which generates random activities for users, which may be added to the calendar at the users discretion.
  • We also utilize the SeatGeek API to generate a list of events by location, calculated by longitude & latitude coordinates.
  • Finally, we utilize the OpenWeatherMap Geocoding API to retrieve the coordinates of a city by name. We pass these geographical coordinates into the SeatGeek API to retrieve a list of events in any particular city.
